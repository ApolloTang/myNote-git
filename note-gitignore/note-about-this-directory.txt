Note on this directory

Directory "deep-dir-selective-ignore-2" is the right way to manage a git directory using gitignore 

never try to managed the entire complex directory all at once like that in 
	"deep-dir-selective-ignore-1"
	"deep-dir-selective-ignore-0"
Add one directory path at a time when you want to negate an ignore.
